# superTools

一个不实用工具集合项目，包含多个有趣且不实用的小工具。

## 项目结构

- `智商检测器/` - 一个有趣的智商测试工具

## 工具列表

### 智商检测器
一个基于Python开发的智商测试应用程序，提供多种测试题目和评分机制。

详细信息请查看：[智商检测器/README.md](智商检测器/README.md)
